# Генератор синтетического датасета фитнес-площадок

Этот инструмент создаёт синтетические изображения фитнес-площадок с разметкой bounding box для различных тренажёров.

## Структура проекта
- `generate_dataset.py` – основной скрипт.
- `requirements.txt` – зависимости.
- `refs/` – папка с референсными изображениями тренажёров. Внутри должны быть подпапки с названиями объектов (см. список `ALL_OBJECTS` в скрипте). В каждой подпапке – несколько изображений (JPEG/PNG) данного тренажёра на простом фоне (или вырезанные).
- `generated/` – папка, куда будут сохраняться сгенерированные изображения (создаётся автоматически).
- `dataset.json` – файл с разметкой (создаётся автоматически).

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости: pip install -r requirements.txt

## Запуск
1. Зайдите в generate_dataset.py и там в начале файла настройте конфигурацию (кол-во изображений на генерацию и т.д.)
2. Выполните python generate_dataset.py
